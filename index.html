OpenHomity

Extensible Open Source Home Automation Suite

Welcome to the OpenHomity project!

OpenHomity is an open source home automation suite capable of controlling a variety of objects, including -

  • Switches - light or otherwise
  • Sensors - digital or analog
  • Garage Doors (coming soon)
  • Alarm Systems (coming soon)
  • Camera Systems (coming soon)

This project is written such that each of the above objects is abstracted from any specific vendor or implementation, so new devices can be supported through pluggable drivers written by users or vendors themselves.

Getting started

What you'll need

  • A hardhat
    • This project is in very early stages, so there is some assembly required
  • A linux server
    • This can be any type of server or distribution you like
    • The code is lightweight enough to run fine on a Raspberry Pi
  • One or more "spoke" devices
    • Arduino Uno ethernet

Setting it up

$ install couchdb
$ setup couchdb
$ git clone openhomity-hub
$ modify config file
$ start hub
$ git clone openhomity-client
$ modify config file
$ alias
